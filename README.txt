Este proyecto se realizo con java.
La clase principal que ejecuta el programa se llama "Juego".
La ejecucion del programa se hace mediante consola.

NOTA: Es muy importante no correr el programa mientras se juega.
Ya que si no la base de datos mostrara conflictos ya que habrian
varios procesos intentando acceder a la misma base de datos. Es 
mejor esperar a terminar la partida o parar el proceso completamente 
antes de volver a correr el programa.

EJECUCION DEL PROGRAMA
------------------------------------------
Durante le ejecucion el programa te preguntara diferentes cosas. Para responder
a las preguntas deberas ingresar un numero entre 1 y 4 que corresponden a las opciones
de la pregunta.

El programa tambien te preguntara constantemente si quieres abandonar el juego hasta donde has llegado.
Para responder a esto se debe ingresar el numero 1 para decir que si, o el numero 2 para decir que no.

por ultimo el programa te preguntara tu nombre y cedula. Esto es para generar un identificador unico
en la base de datos y de esa manera solo actualizar los datos tuyos y no los de otros jugadores.

Finalmente el programa te mostrara el premio que ganaste en esa partida. y el acumulado historico de todas
tus partidas.
